GitHub Gist 云端持久化存储

在电脑导入数据，在手机或异地设备随时读取与同步

同步失败常见原因排查：
  • 本地无数据推送： 如果本地列表为空，点击“推送云端”会被拦截。请先导入 Excel 或手动新增一条数据。
  • Token 权限受限： 生成 Token 时必须勾选 gist 复选框，否则 GitHub API 返回 401/403 错误。
  • 网络连接限制： 请求 api.github.com 接口需确保网络连接通畅（如有代理工具请检查规则）。
  • 格式解析错误： 如果 Gist 是新建的空文档，请先在本地导入或新增数据后点击“将本地推送云端”建立初始结构。
